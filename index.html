Website ini memberikan informasi mengenai diri saya berupa profil data diri dan juga mengenai hobi. Informasi yang akan disampaikan disini akan dijelaskan secara detail agar pengunjung web ini mengenal saya lebih dekat. Profil data diri yang akan ditampilkan berupa nama,tempat tanggal lahir, alamat, dan sebagainya. Kemudian untuk hobi yang akan ditampilkan disini berupa cerita, yang mana menceritakan salah satu pengalaman terbaik dari hobi saya.

Perancangan informasi data diri dan hobi berupa web design ini mencakup segala kalangan karena website berisikan mengenai informasi dan website dapat diakses dengan sangat mudah. Website bisa diakses kapan pun tanpa ada batasnya waktu baik pria ataupun wanita.

Website design informasi sudah dapat diakses sejak website ini diupload pada github dan dapat diakses tanpa batas waktu mengenai informasi profil data diri dan hobi. Melihat dari data pengunjung web yang memiliki hobi sama dengan saya(travelling) biasanya melakukan pencarian mengenai travelling pada waktu-waktu sengang dan tak menentu dan demikian website ini dibuat dan bisa dikunjungi tanpa batas waktu.

Target perancangan website design informasi ini tertuju pada Yang Terhormat Bp. Dosen Anggar Riskinanto, juga ditujukan kepada semua kalangan umum. Website ini bisa dijadikan sebagai dasar gambaran bagi siapapun yang ingin membuat website design sederhana. Website ini juga ditujukan kepada traveller yang sedang mencari rekomendasi tempat travel baru, dan penikmat cerita travelling.

Perancangan web design ini bertujuan untuk memberikan informasi mengenai profil data diri dan hobi, juga penilaian akhir mata kuliah bahasa markup. Kemudian perancangan web design ini merupakan pengimplementasian pembelajaran bahasa markup selama satu semester ini, hasil dari web yang akan ditampilkan ini akan menunjukan sejauh mana saya memahi/mengerti mata kuliah bahasa markup.

Website design dirancang semenarik mungkin sesuai dengan konsep yang saya pegang komunikasi visual untuk menarik minat pengunjung web, dengan konsep sederhana but elegent dengan tujuan mudah dipahami dan baik untuk dipandang
